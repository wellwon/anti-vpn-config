# ANTIVPN

Лёгкое десктоп-приложение: когда активен зарубежный VPN, оно направляет выбранные
российские сайты **напрямую** (мимо туннеля), чтобы они работали полностью — с картинками
и всем содержимым, как будто VPN выключен.

> ⚠️ Сборка пока без нотаризации Apple. macOS покажет «приложение повреждено» — это
> Gatekeeper, а не реальная поломка. Открыть (один раз после установки в «Программы»):
> ```sh
> xattr -dr com.apple.quarantine /Applications/ANTIVPN.app
> ```
> затем открыть обычным способом. (Нотаризация уберёт этот шаг.)

## Обновления
Приложение само проверяет новые версии (этот репозиторий, `latest.json`) и предлагает
обновиться уведомлением по центру экрана — скачивание и перезапуск автоматические.

## Содержимое репозитория (публичное)
- `presets.json` — списки сайтов и сетевые блоки владельцев (автообновляются в приложении);
- `latest.json` — манифест апдейтера;
- **Releases** — готовые сборки приложения.

Исходный код приложения — в приватном репозитории.
