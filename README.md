# anti-vpn-config

Публичные конфигурации для приложения **ANTIVPN** (авто-обновляются клиентом).
Код приложения здесь не публикуется — он в приватном репозитории.

| Файл | Назначение |
|------|------------|
| `presets.json` | Списки сайтов по категориям (Государственные, Маркетплейсы, Банки, ВЭД, Медиа, Соцсети, WeChat) |
| `announcements.json` | Уведомления и кросс-промо для пользователей |
| `latest.json` | Манифест авто-обновления (tauri-plugin-updater) |

Приложение тянет эти файлы по `raw.githubusercontent.com/.../main/*.json` периодически.
Бинарные релизы (`.dmg`/`.exe`) — во вкладке **Releases**.
